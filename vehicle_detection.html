Vehicle Detection

To determine occurrence of a road accident, the system needs to be able to identify the vehicles which is the basic task towards resolving this problem. We will use object detection method to detect the vehicles.



Mask Region based Convolutional framework is used for object detection.



Mask R-CNN is a variant of Deep Neural Network to detect objects in images and generate high quality segmentation mask. It was developed on top of Faster R-CNN (Region based convolutional neural network).

Faster R-CNN generates two output ie. Class label and a bounding box whereas Mask R-CNN in addition to class label & bounding box also generates additional mask.

Stage 1: First stage is Region Proposal Network, which proposes candidate object bounding boxes.
Stage 2: Second stage predicts class, box offset and a binary mask. It uses ROAlign to locate important areas of feature map.

Mask Representation: "A mask encodes an input object's spatial layout. Thus, unlike class labels or box offsets that are inevitably collapsed into short output vectors by fully-connected (fc) layers, extracting the spatial structure of masks can be addressed naturally by the pixel-to-pixel correspondence provided by convolutions."

RoIAlign: Extraction of a small feature map from Region of Interest. Quantization generates misallignments between RoI & extracted features. Hence RoIAlign was introduced.

In image segmentation, a digital image is partitioned into multiple segments and these segments will be used to locate objects in those images based on their boundaries.

Mask R-CNN has two type of image segmentation:

1. Semantic Segmentation: Also known as background segmentation, separates subjects from background in an image. It classifies pixel into fixed groups and does not differentiate between individual instances.
2. Instance Segmentation: It correctly detects all objects in image while correctly segmenting each instance.



Mask R-CNN generates three outputs for each object, a class label, a bounding box and object mask.

In this solution, Mask R-CNN uses instance segmentation to detect cars.

This step of vehicle detection generates an output dictionary of class IDs, bounding boxes, detection scores and masks generated by Mask R-CNN for each video frame.
